# Retrospectives

Telescope runs retrospectives for every closed **milestone** and every closed **issue** that materially changes product/architecture/testing.

## Rule
When an issue or milestone is closed, add a short retro markdown file here and link it from the issue/milestone.

## File naming
- Issues: `ISSUE-<number>-<slug>.md`
- Milestones: `MILESTONE-<name-or-number>-<slug>.md`

## Template
Use this template (keep it short):

```md
# Retro: <Issue/Milestone title>

- Link: <URL>
- Date closed: YYYY-MM-DD
- Owner(s): @user

## What shipped
- 

## What went well
- 

## What went wrong / surprised us
- 

## Lessons / decisions
- 

## Follow-ups
- [ ] <issue/link>
```
